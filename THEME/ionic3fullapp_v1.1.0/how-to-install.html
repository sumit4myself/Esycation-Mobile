Getting started

1.Installation

cd into the directory and then run npm install command.

$ npm install

2.Configuration

Before run ionic serve

1.Create a new firebase project

2.Open the Database section. Click DATA tab > Import JSON > firebase_DATA.json

3.Enable Email/Password sign-in

    -Open the Authentication section.

    -On the SIGN-IN METHOD tab, enable the Email/password sign-in method and click Save.

4.Open the Database section. Click RULES tab

******* insert this code *******

{
  "rules": {
    ".read": "true",
   ".write": "auth != null"
  }
}

5.Update Firebase configuration in /src/app/app.module.ts

********** firebase configuration ***********

export const config = {
   apiKey: your-key,
   authDomain: your-project-authdomain,
   databaseURL: your-database-URL,
   projectId: your-project-id,
   storageBucket: your-storage-bucket,
   messagingSenderId: your-messaging-sender-id
};

3.Run

Testing your app in a browser is as simple as running the serve command in your projectâ€™s root folder.

$ ionic serve


Usage

Theme

-Copy variables.scss (src/theme/variables.scss) to your src/theme folder

-Usage : color="color name"

EXAMPLE :

<ion-navbar color="lightblue" ></ion-navbar>
<button ion-button color="bluegray" >Button</button>

Google map

Add the Google Maps JavaScript SDK

- Include the Google Maps API in your src/index.html file

- Replace Google Maps API key (To generate an API key, go to this page and follow the instructions.)

<script src="http://maps.google.com/maps/api/js?key=YOUR_API_KEY_HERE"></script>

Add the Geolocation plugin

$ npm install --save @ionic-native/geolocation

Image gallery

run npm install command.

npm install ionic-gallery-modal --save

More detail

Chart

run npm install command.

$ npm install ng2-charts --save
$ npm install chart.js --save

Rating

run npm install command.

$ npm install --save ionic2-rating

More detail

Animation

-Copy animation.scss (src/theme/animation.scss) to your src/theme folder

-Import '../theme/animations' in your src/app/app.scss file

@import '../theme/animations';

-Usage : class="effect name"

EXAMPLE :

<button ion-button class="slide-up" >Button</button>
